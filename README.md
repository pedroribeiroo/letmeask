<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/pedroribeiroo/letmeask.git

# Go to server folder
$ cd letmeask

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start

# Access localhost
http://localhost:3000
```
